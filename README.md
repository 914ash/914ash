# 914ash

I build geospatial software, monitoring tools, and workflow systems.

This profile is the short version. If you are scanning for real product work, start with the three repos below.

## Featured Work

### Traffic Cams
`traffic-cams` is a traffic-camera monitoring app built on public feeds. It maps camera coverage and lets a reviewer open one location quickly instead of searching agency sites.

Why open it: it shows data cleanup and a clear interface in the same repo.

### Satellite Challenge
`satellite-challenge` is a tracking dashboard for maritime and aviation activity. It combines live feeds, replay controls, and entity review in one map-driven app.

Why open it: it is the strongest end-to-end product in this public set.

### Symphony
`symphony` is a TypeScript fork of the Symphony workflow service. The work here focuses on visible run state, safer completion rules, and clearer control over autonomous execution.

Why open it: it shows how I approach AI systems when trust and supervision matter more than novelty.

## What I Work On

- geospatial interfaces and monitoring software
- realtime systems that turn incoming data into usable interfaces
- AI workflow and agent infrastructure with visible state and controllable execution
