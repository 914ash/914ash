# Featured Project Blurbs

## Traffic Cams

`traffic-cams` explores how public infrastructure feeds can be turned into a mission-control style geospatial interface. The project combines a websocket-fed backend with a 3D globe UI and a camera inspector that surfaces live feeds and metadata for rapid operator triage.

## Satellite Challenge

`satellite-challenge` is a tactical dashboard for realtime maritime and aviation awareness. It focuses on feed normalization, replay, and operator-facing map controls rather than a generic consumer map experience.

## Symphony

`symphony` highlights my work on frontier AI workflow orchestration. The repo centers on verification-aware completion, observability, and the operational controls needed to use autonomous coding systems responsibly.

## Attractor

`attractor` is a graph-driven AI pipeline engine for orchestrating complex model and agent workflows. It shows how I think about frontier AI systems as controllable infrastructure rather than isolated prompt experiments.
