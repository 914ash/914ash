# Featured Project Notes

## Traffic Cams

`traffic-cams` is a traffic monitoring app built around public camera feeds. It collects data from multiple sources, plots those cameras on a map, and gives the user a quick way to open a single location and inspect the feed details.

Good repo for: product thinking, geospatial UI work, and the problem of making uneven public data usable.

## Satellite Challenge

`satellite-challenge` is a live tracking dashboard for maritime and aviation data. The app normalizes multiple feeds, keeps replay available for review, and presents the result in a map-driven interface that feels operational instead of generic.

Good repo for: full-stack application work, feed normalization, and clear operator-facing interface design.

## Symphony

`symphony` is a TypeScript implementation fork of the Symphony workflow service. The focus here is visible state, safer completion logic, and tighter control around autonomous runs and external-system updates.

Good repo for: workflow systems, AI operations, and practical trust controls in automation software.

### Attractor

`attractor` is a Python engine for graph-based AI workflows. It matters less as a first-stop repo for recruiters, but it is useful if the reader wants to see the underlying orchestration ideas in more detail.

Good repo for: workflow infrastructure, graph execution, and agent-system design.
