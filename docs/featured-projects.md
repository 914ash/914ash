# Featured Project Notes

## traffic-cams

`traffic-cams` is a traffic monitoring app built around public camera feeds. It cleans and normalizes feed payloads, maps coverage, and lets a reviewer jump quickly from region-level view to camera-level inspection.

Why it matters: it demonstrates geospatial UI work and realtime ingest cleanup in the same repo.

## satellite-challenge

`satellite-challenge` is a live maritime and aviation tracking dashboard with replay and entity review. The API and web app share contracts so the operational surface stays consistent with the ingest model.

Why it matters: it is the strongest end-to-end operator dashboard in this set.

## symphony

`symphony` is a TypeScript workflow-service fork focused on observability, verification-aware completion, and controlled autonomous execution.

Why it matters: it shows practical trust controls for long-running AI workflow systems.

## agentic-stories-to-symphony

`agentic-stories-to-symphony` extends the execution model with a planning front-end: intake, canonical spec revisions, approval gates, and deterministic Linear projection before handoff to execution.

Why it matters: it shows disciplined planning-to-execution pipeline design, not just runtime orchestration.

## dprk-document-analysis-app

`dprk-document-analysis-app` is a sanctions-intelligence monorepo with analyst-facing dashboard views plus entity-resolution and network-drift analysis packages, including explicit upstream provenance notes.

Why it matters: it adds defense-adjacent research tooling and method transparency to the portfolio.
